# 《阴契》第一幕：归魂村纸新娘案垂直切片

## 收口原则

本阶段不新增大引擎，只把已有引擎收口到一个可玩的案件样板：

```text
归魂村村口 → 河桥 → 老宅 → 戏台 → 封氏祠堂
```

目标是证明：

```text
地点皮肤
→ 可交互物
→ 线索链
→ 民俗规则
→ 恐怖导演
→ 动态渲染
→ 开放剧情线
→ 第一幕结局钩子
```

能够完整跑通。

## 世界观核心

恐怖复苏不是鬼从地府爬出，而是“规矩”获得实体约束力。

黄历宜忌、节气、丧葬禁忌、扎纸祖训，原本只是习俗；在第一道裂缝阶段，它们变成了物理因果律。违禁即触发灾祸。

归魂村是第一批重灾区，因为它偏远、封闭、宗族祭祀完整，旧时代规矩还在被日常执行。

## 主角

陈默，24岁，深州市深夜跑腿外卖员。

性格关键词：

```text
务实 / 冷漠 / 对时间和数字敏感 / 第一反应是算成本和生路
```

误入原因：

```text
2000元高额跑腿单
→ 子时前送白蜡烛和黄纸到归魂村老祠堂
→ 大雾 / 导航死机 / 电瓶车电量归零
→ 推车撞见归魂村界碑
```

## 主案真相

林晚秋，21岁，美术系学生，被拐卖到归魂村，强行配阴婚。

村长封建德为了给暴毙的傻儿子配阴婚，买来林晚秋。阴婚当夜，林晚秋反抗，被村民以香油白布缠住口鼻，推入后山寒泉溺死。死后被九叔制成纸新娘。

因为世界规则松动，纸新娘成为归魂村恐怖法则源头。

## 核心规则闭环

### 子时不可点睛

点睛即赋予人格。

子时用活人血或朱砂为纸新娘点睛，会解开禁锢并反噬点睛者，使其纸质化。

对应规则：

```text
midnight_paper_eye_taboo
```

### 族谱解契

封氏族谱是宗族气运枷锁。林晚秋之名被强行写入封家偏房名分，使她受封氏宗族压制。

解法不是直接攻击纸新娘，而是：

```text
找到《扎纸遗书》
→ 得知族谱为锁
→ 在祠堂看到林晚秋血名
→ 用白蜡烛引路业火烧毁族谱
→ 名分消散，阴阳锁自解
```

对应规则：

```text
genealogy_contract_burn_rule
```

## 五地点线索链

### 1. 村口

作用：开场、切断退路。

关键线索：

```text
苏青青备用麦克风 → 听到落水声 → 引向河桥
```

### 2. 河桥

作用：确认死因和身份。

关键线索：

```text
香油白布
学生证项链
草鞋脚印
```

### 3. 老宅

作用：确认宗族犯罪与村长主谋。

关键线索：

```text
红嫁衣
地下室铁链
阴婚聘书
村长账本
```

### 4. 戏台

作用：揭示规则和拿到祠堂入口。

关键线索：

```text
《扎纸遗书》
皮影戏幕布
祠堂后门钥匙
```

### 5. 祠堂

作用：最终对抗和第一幕终章。

关键线索：

```text
纸新娘
封氏族谱
朱砂笔
白蜡烛
```

## 第一幕结局

陈默用跑腿送来的白蜡烛点燃封氏族谱，族谱上的宗族契约被烧断。

封氏族人因血脉连坐被法则反噬，纸质化后随灰烬崩散。

林晚秋不再被封氏名分压制，纸新娘转身沉入寒泉。

陈默逃出村后，手机恢复信号，但看到的是全国各地“规矩复苏”的新闻推送。

结论：

```text
不是归魂村闹鬼。
是整个世界的规则正在死掉。
```
